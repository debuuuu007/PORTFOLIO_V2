# Portfolio Website

A modern, responsive personal portfolio built with React, Vite, and Tailwind CSS. It highlights the developer’s services, skills, and contact information in a polished single-page experience.

## Features

- Responsive navigation and mobile menu
- Hero section with strong introduction and call-to-action
- Services, About, and Contact sections
- Dark theme with orange-accented visual design
- Working contact form powered by EmailJS

## Tech Stack

- React
- Vite
- Tailwind CSS
- EmailJS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

To create a production build:

```bash
npm run build
```
